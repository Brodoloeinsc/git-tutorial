Git Helps



Base codes


Adicionar um repositorio remoto

git remote add nome_de_preferencia link_do_repositorio

É interessante usar o nome origin

Se for adicionar todos os arquivos

git add *


Caso não vá adicionar todos

git add nome_do_arquivo.extensao


"Salvar" os arquivos

git commit -m "Descrição do arquivo"


Selecionar a Branch

git branch -M "nome da Branch"


Para upar para o github

git push -u nome_do_remoto nome_da_branch
